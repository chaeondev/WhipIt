#### 일상의 스타일을 공유하고 해시태그 기반 검색이 가능한 커뮤니티 형식의 SNS 앱입니다.

## 프로젝트 소개

> 앱 소개
- 회원가입, 로그인 기능 제공
- 사진과 텍스트를 기반한 게시글 작성, 편집 및 삭제 가능
- 게시글에 해시태그 추가 가능, 해시태그 기반으로 게시글 필터링 기능 제공
- 북마크, 댓글을 통해 다른유저와 소통 가능
- 다른 유저를 팔로우하고 프로필 확인 가능
- 나의 프로필에서 게시글, 팔로우 관리 기능 제공
